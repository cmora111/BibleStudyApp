# Ultimate Bible App v8.2 User Guide

## What v8.2 adds

Version 8.2 adds a starter Strong's dataset bundle to help you test scholar workflows immediately.

Included:
- demo Strong's lexicon CSV
- demo alignment CSV
- scholar search engine
- scholar token demo loader
- book-name normalization from v8.1

## What you can do immediately

### Strong's lookup
Use the Strong's panel with codes like:
- `G25`
- `G4903`
- `H7225`

### Scholar search
Use structured search:
- `strongs:G25`
- `lemma:agapao`
- `morph:V-RAI-1P`

### Reliable verse lookup
Type:
- `galations`
- `gal`
- `rom`
- `rev`

and the UI can normalize them to the canonical book names once the v8.1 patch is applied.

## Demo workflow

1. Load the demo scholar tokens.
2. Open Romans 8:28 in ESV.
3. Run scholar search for `strongs:G25`.
4. Click tagged words if your UI has the scholar token wiring applied.
