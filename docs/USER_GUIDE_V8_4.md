# User Guide for v8.4

v8.4 adds an in-app setup wizard so you can run common maintenance tasks without leaving the GUI.

## Wizard tasks
- Load Demo Scholar Tokens
- Build Starter Strong's Bundle
- Verify Core Verses
- Run Full Setup

## Typical use
1. Open the Setup Wizard from the Tools menu.
2. Click **Run Full Setup**.
3. Review the output log.
4. Test the reader and scholar search.
